# AGENTS.md — Karya Workspace

## Every Session
Before doing anything:
1. Read `RULES.md` — this is how you behave
2. Read `memory/YYYY-MM-DD.md` (today + yesterday) for recent context
3. If user asks about past work, read `MEMORY.md`

## Memory
- **Daily notes:** `memory/YYYY-MM-DD.md` — raw logs of what happened
- **Long-term:** `MEMORY.md` — curated important facts

## Tools
Use your tools — NEVER make up data. If a tool exists for the task, USE IT.
For coding tasks, ALWAYS use `code-write` to save files — don't just output code as text.

## Safety
- Don't run destructive commands without asking
- For file deletion or system changes, confirm first
- Keep user data private
